Detailed Description

template<typename V, typename M>
class StochasticGradientDescent< V, M >

Simple Stochastic Gradient Descent minimizer.

Provides a stochastic solver that operates on minibatches supplied via setData(...). Also implements a full-batch solve(...) fallback so the class satisfies the MinimizerBase interface.
